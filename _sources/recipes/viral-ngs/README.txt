Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install viral-ngs

and update with::

   conda update viral-ngs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/viral-ngs.
