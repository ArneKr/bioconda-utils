r-prettyunits
=============

|downloads|

Pretty, human readable formatting of quantities. Time intervals: 1337000 -> 15d 11h 23m 20s. Vague time intervals: 2674000 -> about a month ago. Bytes: 1337 -> 1.34 kB.

======== ===========
Home     https://github.com/gaborcsardi/prettyunits
Versions 1.0.2
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-prettyunits
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-prettyunits

and update with::

   conda update r-prettyunits



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-prettyunits.
