.. _`r-scales`:

r-scales
========

|downloads|

Graphical scales map data to aesthetics, and provide methods for automatically determining breaks and labels for axes and legends.

======== ===========
Home     https://github.com/hadley/scales
Versions 0.4.1
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-scales
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-scales

and update with::

   conda update r-scales



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-scales.
