r-dplyr
=======

|downloads|

A fast, consistent tool for working with data frame like objects, both in memory and out of memory.

======== ===========
Home     https://github.com/hadley/dplyr
Versions 0.5.0
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-dplyr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-dplyr

and update with::

   conda update r-dplyr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-dplyr.
