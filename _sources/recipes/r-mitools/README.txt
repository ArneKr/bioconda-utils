r-mitools
=========

|downloads|

Tools to perform analyses and combine results from multiple-imputation datasets.

======== ===========
Home     
Versions 2.3
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-mitools
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-mitools

and update with::

   conda update r-mitools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-mitools.
