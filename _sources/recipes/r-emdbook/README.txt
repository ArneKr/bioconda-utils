r-emdbook
=========

|downloads|

Auxiliary functions and data sets for "Ecological Models and Data", a book presenting maximum likelihood estimation and related topics for ecologists (ISBN 978-0-691-12522-0).

======== ===========
Home     https://CRAN.R-project.org/package=emdbook
Versions 1.3.9
License  GPL-2 | GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-emdbook
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-emdbook

and update with::

   conda update r-emdbook



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-emdbook.
