mmtf-python
===========

|downloads|

A decoding libary for the PDB mmtf format

======== ===========
Home     http://mmtf.rcsb.org
Versions 1.0.2, 1.0.5
License  Apache-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/mmtf-python
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mmtf-python

and update with::

   conda update mmtf-python



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/mmtf-python.
