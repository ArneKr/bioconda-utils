dscr
====

high-performance compression of sequencing reads stored in FASTQ format

======== ===========
Home     https://github.com/lrog/dsrc
Versions 2014.12.17
License  GNU GPL 2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/dsrc
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dscr

and update with::

   conda update dscr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/dscr.
