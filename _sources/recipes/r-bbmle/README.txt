r-bbmle
=======

|downloads|

Methods and functions for fitting maximum likelihood models in R. This package modifies and extends the 'mle' classes in the 'stats4' package.

======== ===========
Home     https://CRAN.R-project.org/package=bbmle
Versions 1.0.18
License  GPL-2 | GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-bbmle
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-bbmle

and update with::

   conda update r-bbmle



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-bbmle.
